## 课文语法

```js
// 【Key structure:】
// 现在进行时
// 一般现在时
// 感叹句
```

```js
// 【时态:】
// 现在的事，正在进行
// 体现：谓语动词变成对应的公式
// 一共16种时态 (四种时间：过去、现在、将来、过去将来；四种状态：一般、进行、完成、完成进行)
```

```js
// 【一般现在时:】
// 谓语v. = 原型/第三人称单数(do/does)
// --- 第三人称单数主语：he she it
// --- 可数名词单数
// --- 不可数名词
// 1. I tell you a secret.
// 2. He tells you a secret.
// 3. Your friend tells you a secret.
```
