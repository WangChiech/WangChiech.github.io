## 选择器

> css 选择器，找到元素添加**样式** (.class)
>
> jQuery 选择器，找到元素添加**行为** ($('.class'))

1. `$('#id')` 

2. `$('.class')` 

3. `$('div')` 

4. `$('*')` 

5. `$('div, span, .class')` 并集选择器

6. `$('form input')`  (form 下的所有 input 标签)

   ```html
   <form>
     <label>Name:</label>
     <input name="name" />  **
     <fieldset>
         <label>Newsletter:</label>
         <input name="newsletter" />  **
    </fieldset>
   </form>
   <input name="none" />
   ```

7. `$("form > input")` 子代选择器

   ```html
   <form>
     <label>Name:</label>
     <input name="name" /> **
     <fieldset>
         <label>Newsletter:</label>
         <input name="newsletter" />
    </fieldset>
   </form>
   <input name="none" />
   ```

8. prev + next

   - prev ，任何有效选择器
   - next，prev 后面紧邻的第一个同辈选择器

   ```html
   <form>
     <input name="front" />  
     <label>Name:</label>
     <input name="name" />  **
     <fieldset>
         <label>Newsletter:</label>
         <input name="newsletter" />  **
    </fieldset>
   </form>
   <input name="none" />
   ```

   ```js
   $("label + input")
   // 结果 [ <input name="name" />, <input name="newsletter" /> ]
   ```

9. prev ~ siblings (匹配 prev 元素之后的所有**同辈** siblings 元素)

   ```html
   <input name="front"/>
   <form>
     <label>Name:</label>
     <input name="name" />
     <fieldset>
         <label>Newsletter:</label>
         <input name="newsletter" />
    </fieldset>
   </form>
   <p>
       <input name="p">
   </p>
   <input name="none" />  **
   ```

   ```js
   $("form ~ input")
   // 结果 [ <input name="none" /> ]
   ```

10. `$('li:first')` (获取第一个元素)

    ```html
    <ul>
        <li>list item 1</li>  **
        <li>list item 2</li>
    ```

11. `$("input:not(:checked)")` (查找所有未选中的 input 元素)

    ```html
    <input name="apple" />
    <input name="flower" checked="checked" />  **
    ```

12. `$("tr:even")` (匹配所有索引值为偶数的元素，从 0 开始计数)

    ```html
    <table>
      <tr><td>Header 1</td></tr>  **
      <tr><td>Value 1</td></tr>
      <tr><td>Value 2</td></tr>  **
    </table>
    ```

13. `$("tr:odd")` (匹配所有索引值为奇数的元素，从 0 开始计数)

    ```html
    <table>
      <tr><td>Header 1</td></tr>
      <tr><td>Value 1</td></tr>  **
      <tr><td>Value 2</td></tr>
    </table>
    ```

14. `$("tr:eq(1)")` (查找索引为 1 的 tr)



16. `$("tr:lt(2)")` (查找索引**小于** 2 的 tr)


## DOM 操作

**创建节点** 

1. `$('<p>创建p标签<p>')` 

   ```js
   var $p = $('<p class="p">创建p标签<p>')
   ```

**插入节点** 

2. `$("p").append("<b>Hello</b>")` 

   ```html
   <p>I would like to say: </p>
   <!--结果: [ <p>I would like to say: <b>Hello</b></p> ]-->
   ```

3. `$('<p></p>').appendTo('div')` 

   ```html
   <div></div>
   <!--结果: [ <div><p></p></div> ]-->
   ```

4. `$("p").prepend("<b>Hello</b>")` 

   ```html
   <p>I would like to say: </p>
   <!--
   	结果: 
   	[ 
   		<p>
               <b>Hello</b>
               I would like to say: 
           </p> 
   	]
   -->
   ```

5.  `$("p").prependTo("#foo")` (把所有段落追加到ID值为foo的元素中)

   ```html
   <p>I would like to say: </p><div id="foo"></div>
   <!--
   	结果: 
   	[ 
   		<div id="foo">
   			<p>I would like to say: </p>
   		</div>
   	]
   -->
   ```

6. `$("p").after("<b>Hello</b>")` 

   ```html
   <p>I would like to say: </p>
   <!--结果: <p>I would like to say: </p><b>Hello</b>-->
   ```

7. `$("p").before("<b>Hello</b>")` 

   ```html
   <p>I would like to say: </p>
   <!--结果: [ <b>Hello</b><p>I would like to say: </p> ]-->
   ```
