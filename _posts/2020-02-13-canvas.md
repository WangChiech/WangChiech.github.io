canvas 是HTML提供的新标签，矩形画布，可以用 JavaScript 控制每一像素在上面进行绘画，拥有绘制路径、矩形、圆形等。

canvas 可以应用于游戏、可视化数据（数据图表，如百度的echar)、banner 广告、未来可能会有模拟器（航天航空等）、远程计算机控制、图形编辑器（Photoshop网页版）、完整的canvas移动化应用.....

```html
<!-- 设置canvas的宽高是通过canvas的内部属性来设置的，通过css设置的宽高会拉伸canvas -->
<!-- 重新设置 canvas 标签的宽高属性会让画布擦除所有的内容 -->
  <canvas id="demo" width="600" height="600">
    ie9 以上才支持
    你的浏览器不支持 canvas
    <!-- flash -->
  </canvas>
  <script>
    // 1. 拿到canvas标签
    var canvas = document.getElementById('demo');
    canvas.style.border = '1px solid red';
    // canvas.width = 600; 
    // canvas.height = 600;
    // canvas.style.height = 1200 // 错误，不使用css设置canvas样式

    // 2. 拿到canvas的上下文
    var context = canvas.getContext('2d'); // 'webgl'

    // 3. 设置
    context.lineWidth = 4; // 设置线宽
    context.strokeStyle = 'red';  // 描边颜色
    context.fillStyle = 'green'; // 需在context.fill()前设置才能生效
    context.fill(); // 可以自动闭合路径
      
    // 4. 绘制
    context.moveTo(0,0); // 画笔移动到（100,100）
    context.lineTo(200, 100); // 绘制直线
    context.lineTo(100, 200);
      
    // 5. 闭合路径
    // context.lineTo(0,0) 
    context.closePath() 
    
	// 6. 描边
    context.stroke(); // 描边

  </script>
```

```js
// 开始路径：
ctx.beginPath();
// 闭合路径：
ctx.closePath();
```

```js
// 1. 创建矩形
ctx.rect(x, y, width, height); // x, y是矩形左上角坐标， width和height都是以像素计
// 2. 创建描边矩形
ctx.strokeRect(x, y, width, height);
// 3. 创建填充矩形
ctx.fillRect(x, y, width, height);
// 4. 清除矩形
ctx.clearRect(x, y, width, hegiht); // 清除矩形内绘制的内容
```

```js
// 绘制圆形
ctx.arc(x,y,r,sAngle,eAngle,counterclockwise); // counterclockwise：是否是逆时针
```

```js
// 1. 绘制文本
ctx.fillText(txt,x,y);
// 在画布上绘制文本（无填充）
ctx.strokeText()
// 2. 返回包含指定文本宽度的对象
ctx.measureText()
// 3. 文本对齐方式
ctx.textAlign = 'end';
```

textAlign 设置或返回文本内容的当前对齐方式

- start : 默认。文本在指定的位置开始。
- end : 文本在指定的位置结束。
- center: 文本的中心被放置在指定的位置。
- left : 文本左对齐。
- right : 文本右对齐。

```js
// 绘制图片

};
```

```js
// 1. 绘制图片
context.drawImage(img,x,y,width,height);
// 2.  图片裁剪，并在画布上定位被剪切的部分
context.drawImage(img,sx,sy,swidth,sheight,x,y,width,height); 
// sx,sy 裁剪的左上角坐标，swidth：裁剪图片的高度。 sheight:裁剪的高度
```
