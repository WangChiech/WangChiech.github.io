`sass`基于`Ruby`语言开发而成，因此安装`sass`前需要[安装Ruby](http://rubyinstaller.org/downloads)。[安装操作](https://www.sass.hk/install/)

## 1. 基本使用

### 1.1 使用变量

`sass`使用`$`符号来标识变量，变量的优点是可以反复使用、可表明意义与用途。

#### 1.1.1 变量声明

变量值可以是一个属性值，或是以空格分隔的多个属性值，或以逗号分隔的多个属性值。

变量的使用范围取决于变量的声明位置。

```css
$nav-color: #F90;
nav {
  $width: 100px; // 只能在此规则块使用
  width: $width;
  color: $nav-color;
}

 /* 编译后 */
nav {
  width: 100px;
  color: #F90;
}
```

#### 1.1.2 变量引用

变量是作为取代`CSS` 属性标准值的另一种形式，通过改变变量的值，间接改变所有引用该变量的地方生成的值。

在声明变量时，变量也可以引用其他变量。

```css
$highlight-color: #F90;
$highlight-border: 1px solid $highlight-color;
.selected {
  border: $highlight-border;
}

 /* 编译后 */
.selected {
  border: 1px solid #F90;
}
```

#### 1.1.3 变量名用中划线或下划线

`sass` 变量名使用中划线、下划线两种命名方式命名的内容是可以互相通用的(建议都使用中划线)。

```css
$link-color: blue;
a {
  color: $link_color;
}

 /* 编译后 */
a {
  color: blue;
}
```

### 1.2 嵌套 CSS 规则

为简化`css` 重复写选择器的问题，`Sass` 支持嵌套的形式来表示后代选择器。

```css
#content {
  article {
    h1 { color: #333 }
    p { margin-bottom: 1.4em }
  }
  aside { background-color: #EEE }
}

 /* 编译后 */
#content article h1 { color: #333 }
#content article p { margin-bottom: 1.4em }
#content aside { background-color: #EEE }
```

#### 1.2.1 父选择器的标识符 &

当包含父选择器标识符的嵌套规则被打开时，它不会像后代选择器那样进行拼接，而是`&`被父选择器直接替换：

```css
article a {
  color: blue;
  &:hover { color: red }
}
 /* 编译后 */
article a { color: blue }
article a:hover { color: red }
```

#### 1.2.2 群组选择器的嵌套

```css
.container {
  h1, h2, h3 {margin-bottom: .8em}
}

 /* 编译后 */
.container h1, .container h2, .container h3 { margin-bottom: .8em }
```

```css
nav, aside {
  a {color: blue}
}

 /* 编译后 */
nav a, aside a {color: blue}
```

#### 1.2.3 子组合选择器和同层组合选择器：>、+和~

```css
article {
  ~ article { border-top: 1px dashed #ccc } // 同层选择器(选择所有跟在article后的同层article元素，不管它们之间隔了多少其他元素)
  > section { background: #eee } // 子代选择器
  dl > {
    dt { color: #333 }
    dd { color: #555 }
  }
  nav + & { margin-top: 0 } // 相邻选择器
}
```

#### 1.2.4 嵌套属性

```css
nav {
  border: {
  style: solid;
  width: 1px;
  color: #ccc;
  }
}

 /* 编译后 */
nav {
  border-style: solid;
  border-width: 1px;
  border-color: #ccc;
}
```

```css
nav {
  border: 1px solid #ccc {
  left: 0px;
  right: 0px;
  }
}

 /* 编译后 */
nav {
  border: 1px solid #ccc;
  border-left: 0px;
  border-right: 0px;
}
```

### 1.3 导入 SASS 文件

`sass`的`@import`规则在生成`css`文件时就把相关文件导入进来，无需发起额外的下载请求。另外，所有在被导入文件中定义的变量和混合器均可在导入文件中使用。
