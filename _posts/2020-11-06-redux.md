## Redux原理

> React本身是一个非常轻量级的视图层框架。
>
> React = Reducer + Flux
>
> React 在2013年开源的时候除了放出React，还放出了Flux框架（官方推出的最原始的辅助React使用的数据层框架）

1. store是唯一的
2. 只有store能够改变自己的内容
3. Reducer必须是纯函数（固定的输入一定得有固定的输出，且无副作用）
4. 一个组件只有render函数，建议携程无状态组件（一个函数）

### 安装

调试工具 Redux DevTools

```markdown
1. yarn add redux
2. 创建store文件夹
3. store文件夹下创建index.js
4. store文件夹下创建reducer.js
5. store文件夹下创建actionCreators.js
```

```js
// index.js
import { createStore } from 'redux'
import reducer from './reducer'

const store = createStore(reducer)
export default store
```

```js
// reducer.js
const defaultState = {}
export default (state = defaultState, action) => {
    // store里存储的所有数据
    return state
}
```

```js
// 组件获取store数据
import store from './store'
import { getInputChangeAction } from './store/actionCreators'
...
// 获取store中数据
this.state = store.getState()
// store更新会自动触发
store.subscribe(() => {
    // 将store中更新过的数据设置给state
    this.setState(store.getState())
})
...
handleInputChange (e) {
    const action = {
        type: 'change_input_value',
        value: e.target.value
    }
    // const action = getInputChangeAction(e.target.value)
    // reducer 会接收到action
    store.dispatch(action)
}
```

```js
const { CHANGE_INPUT_VALUE } from './actionTypes'
export const getInputChangeAction = (value) => {
    type: CHANGE_INPUT_VALUE,
        value
}
```
