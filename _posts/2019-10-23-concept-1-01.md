## 课文语法-简单句

- 简单陈述句
  - 陈述一件事的句子，单纯的说出来
  - 其他所有句子演变的基础

- 简单陈述句的构成——核心

  - 名词 + 动词

  - n. + v.

  - 主语 + 谓语 (一主、一谓)

    ```js
    // Birds fly.
    // We walk.
    // 主语 + 谓语v. + ?
    // I have a seat.
    // The play was interesting.
    ```

- 简单陈述句的构成——语序
  
- The policeman arrested the thief
  
- 简单陈述句的变化

  ```js
  // Boys talk.
  // Lovely boys were talking loudly.
  // Yesterday after class lovely boys were talking loudyly about the exam in the classroom.
  // 句子由短变长，先有句子的核心，再加入修饰。
  // I go.
  // I went.
  // I went to the theatre (介词短语表地点)
  // Last week I went to the theatre.
  
  // I have a seat.
  // I had a seat.
  // Ihad a very good seat.
  ```

  - 谓语动词的变化

    一件事中只能包含一个谓语动词

    ```js
    // 谓语动词一共有4中变化(时态、语态、情态、否定)
    // They were talking loudly.
    // It was bougth by my grandfather.
    // I could not hear the actors.
    // I did not enjoy it.
    ```

## 简单陈述句扩展

```js
// 【简单陈述句五类:】
// They meet.  主谓
// I love you. 主谓宾
// I tell you a sercret.  主谓双宾(3课讲)
// I find you interesting.  主谓宾补(4课讲，使用少)
// You are beautiful.  主系表
```

```js
// They meet. 主谓  vi.不及物
// I love you. 主谓宾  vt.及物
```

```js
// You are beautiful.  主系表  系动词(没有实在的动作，起联系的)
// 【系动词三中情况:】
// 1. be 动词(单独出现)
// 2. get become turn go grow "变得"
// 3. look sound smell taste feel "看起来/听起来/闻起来/品尝起来/感觉起来"
```

```js
// 【不规则变化的动词:】
// go(原型) went(过去式，用在一般过去时态) gone(过去分词，可用于完成时态、被动语态、不做谓语做非谓语动词)
// have had had having
// do  did  done 
// be  was/were  been
// sit  sat  sat  sitting
// get  got/gotten  getting
// pay  paid  paid
// bear bore  borne  容忍
// say  said  said
```
