## 插件

### **Babel**	编译工具（转码器）

**官网**

1. http://babeljs.io/
2. https://www.babeljs.cn/		中文

**插件**

1. `npm install babel-cli --save-dev` 

2. `npm install babel-preset-env --save-dev`

3. 在根目录下创建一个`.babelrc`的配置文件（rc，run command）

   ```js
   {
       "presets": ["env"]
   }
   ```

> mac里的隐藏文件显示与否，执行下面命令
> `defaults write com.apple.finder AppleShowAllFiles YES; killall Finder`	显示Mac隐藏文件
> `defaults write com.apple.finder AppleShowAllFiles NO; killall Finder`	隐藏Mac隐藏文件

**编译**

1. 直接编译文件（编译后的结果会在终端里显示出来）
   `babel js/src/index.js	` 

2. 编译并输出结果到单个文件（`-o`为`--out-file`的简写）
   `babel js/src/index.js -o js/src/newIndex.js` 

3. 文件修改后自动编译（`-w`为`--watch`的简写）
   `babel js/src/index.js -w -o js/src/newIndex.js ` 

4. 编译整个目录里的文件（`-d`为`--out-dir`的简写）

   `babel js/src -d js/dist` 

**命令编译**

1. 在`package.json`的`scripts`里添加一条属性，如下

   ```js
   "scripts": {
   	"build": "babel js/src -d js/dist"
   }
   ```

2. 在用到编译的时候执行命令：`npm run build` 

**`babel-polyfill`（垫片）** 

用来转除了语法外的`API`

1. 安装：`npm install --save babel-polyfill`

   注意：这个模块需要在代码运行前就要工作，所以需要放到运行环境中。需要用到`--save`

2. 在用到的代码里最上面引入：`import 'babel-polyfill'`
